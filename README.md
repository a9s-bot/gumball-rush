# Gumball Rush

A colorful browser game built with HTML, CSS, JavaScript, Canvas, and Matter.js physics.

## Run locally

```sh
python3 -m http.server 8000
```

Open `http://localhost:8000`.

## Controls

- Select **Start Round** to begin.
- Select the machine knob or press Space to dispense a gumball.
- Match the current order color to build a scoring streak.
